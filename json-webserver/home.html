Practice with JSON documents and the Fetch

Assignment Overview

This assignment was practice working with JSON documents and the 'Fetch' native Javascript functionality. The data is available here and contains my college education history. This page has two functions:

  • Get All Data - retrieves entire JSON document and prints it to the DOM in the 'Fetch Response' section.
  • Get Data By Index - retrieves a particular item in the JSON document based on the provided value in the 'JSON Object Index' input field.
    • If found - the specific JSON object is printed to the DOM in the 'Fetch Response' section.
    • If not found - a generic error is printed to the DOM in the 'Fetch Response' section.

If there is an error with the Fetch request or if some invalid input is provided, then a generic error is printed to the 'Fetch Response' section. This section also has a status indicator that will show the success/failure of the Fetch request.
